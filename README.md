In this project, we emulate all the following HTTPS status codes:
200 OK
201 Created(the appropriate resource has been created)
400(Bad request - It essentially means the server received the request but found it structurally or syntactically incorrect. )
401(Unauthorized access - the client doesnt have proper access to the server side)
403(Forbidden)
404(Not found - the needed resource cannot be found)
409 conflict
500 Internal Server error
503 Service unavailable
